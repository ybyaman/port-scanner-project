# Port Scanner Project

A simple Python-based port scanner that identifies open common ports on a target system and provides basic security risk analysis for exposed services.

## Features
- Scans common ports (e.g., 21, 22, 80, 443)
- Detects open ports using socket connections
- Provides basic security insights for each open port
- Multi-threaded scanning for faster performance

## Usage

Run the script:

```bash
python3 port_scanner.py
```

Then enter a target hostname or IP address:
```bash
scanme.nmap.org
```
